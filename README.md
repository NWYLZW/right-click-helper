## 介绍
由python pyqt模块开发的一款windows环境下卡片化右键菜单选项管理工具

## 功能点
* [x] 巨好看的UI
* [x] 卡片化的管理右键菜单项
* [ ] 拖拽的方式管理右键菜单
* [x] 备份当前配置，并分享
* [x] 自动检测更新
